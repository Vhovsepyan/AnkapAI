# Ankap Trading Bot

A Java/Kotlin-based algorithmic trading bot using Binance Spot Testnet.

## Features
- Binance Spot Testnet integration
- EMA crossover strategy (configurable)
- Position-aware trading (long-only)
- Daily PnL tracking with max daily loss protection
- Start/Stop REST API
- Web-based dashboard UI

## Tech Stack
- Kotlin, Spring Boot
- Binance Java Connector
- REST API
- HTML/JS dashboard

This project is for educational purposes.
Do NOT use with real funds without extensive testing.
